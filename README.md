# Work Hours Management System

## Features
- **Create**: Add new work hour records.
- **Read**: View existing work hour records.
- **Update**: Modify existing records using the `ID`.
- **Delete**: Remove records by `ID`.

## Technologies Used
- **Java**: Core programming language for logic and GUI.
- **JavaFX**: Framework for building the user interface.
- **Oracle Database**: Used to store the work hours data.
- **JDBC**: Used to connect Java to the Oracle database.
